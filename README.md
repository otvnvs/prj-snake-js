# prj-snake-js

A classic, retro-style Snake game built with vanilla JavaScript and HTML5 Canvas. This project was developed to practice game loop logic, coordinate-based collision detection, and clean code principles.

**[Live Demo Link](https://github.io)**

## Features
- **Classic Gameplay:** Navigate the snake to eat food and grow.
- **Score Tracking:** Real-time score updates as you play.
- **Collision Logic:** Game over triggers if the snake hits the walls or itself.
- **Responsive Design:** Centered, focused layout for an arcade feel.

## Tech Stack
- **HTML5:** For page structure and the Canvas element.
- **CSS3:** For the dark-mode "Matrix" aesthetic.
- **JavaScript (ES6):** For the game engine, movement logic, and rendering.

## Key Learnings
- **The Game Loop:** Implementing `setTimeout` to create a consistent frame rate.
- **Coordinate Systems:** Mapping the snake's body parts to a grid to handle movement and growth.
- **Event Listeners:** Handling keyboard input for smooth directional changes.
